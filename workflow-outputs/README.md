# Workflow Outputs

This folder is the landing zone for artifacts generated by the `requirements-to-userstories` agentic pipeline when run against `../requirements/`.

Expected contents once a pipeline run completes:
- `features/` — decomposed feature definitions (output of Feature Decomposition stage)
- `user-stories/` — Jira-ready user stories (output of User Story Generation stage)
- `clarification-requests/` — any S1.7 clarification packages raised against ambiguous SRs
- `run-logs/` — audit logs / confidence scores from the orchestrator, if exported

This folder is intentionally empty until the first pipeline run. Do not hand-author files here — anything here should be traceable to a specific pipeline run against a specific `requirements/` input.
